# Eventos

Tratamento genérico dos eventos no Gartic

## Build

Para gerar o arquivo compilado e compatível com todos os browsers via babel,
executar o comando:
```
npm run build
```

## Sintaxe

Verificar a sintaxe padrão através do comando:
```
npm test
```
